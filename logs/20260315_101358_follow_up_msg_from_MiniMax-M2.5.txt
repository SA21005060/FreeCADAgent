这是一个使用 FreeCAD Python 编写的宏，用于创建一个漂亮的花瓶。

这个脚本执行以下操作：
1.  创建一个新的 FreeCAD 文档。
2.  在 XY 平面上创建一个草图（Sketch），绘制花瓶的半边轮廓（包含底座、瓶身、瓶颈和瓶口）。
3.  使用“旋转”（Revolution）功能将二维轮廓绕中心轴旋转 360 度，形成三维实体。
4.  设置颜色为赤陶土色（Terracotta）。
5.  自动调整视图为等轴测视图（Isometric View）并适应模型大小。

### FreeCAD Python 宏代码

你可以将以下代码复制到 FreeCAD 的 Python 编辑器中运行，或者保存为 `.py` 文件然后在 FreeCAD 中执行。

```python
import FreeCAD
import Part
import Sketcher
import PartDesign

# --- 1. 创建新文档 ---
doc = FreeCAD.newDocument("Vase_FreeCAD")
App.setActiveDocument("Vase_FreeCAD")
App.ActiveDocument = App.getDocument("Vase_FreeCAD")
Gui.ActiveDocument = Gui.getDocument("Vase_FreeCAD")

# --- 2. 创建实体 Body ---
body = App.ActiveDocument.addObject('PartDesign::Body', 'VaseBody')

# --- 3. 创建草图 (绘制花瓶轮廓) ---
# 在 XY 平面上创建草图
sketch = App.ActiveDocument.addObject('Sketcher::SketchObject', 'Profile')
sketch.Support = App.ActiveDocument.XY_Plane
sketch.MapMode = 'FlatFace'

# 定义轮廓点 (X, Y) - X 为半径，Y 为高度
# 这些点定义了花瓶的半边截面形状
profile_points = [
    FreeCAD.Vector(0, 0, 0),       # 底部中心
    FreeCAD.Vector(20, 0, 0),     # 底部边缘
    FreeCAD.Vector(50, 30, 0),    # 瓶身最宽处
    FreeCAD.Vector(15, 65, 0),    # 瓶颈收缩处
    FreeCAD.Vector(22, 80, 0),    # 瓶口外翻
    FreeCAD.Vector(0, 80, 0)      # 顶部中心
]

# 将点连成线段添加到草图
# 注意：必须确保轮廓是闭合的或连接到旋转轴上
for i in range(len(profile_points) - 1):
    line = Part.LineSegment(profile_points[i], profile_points[i+1])
    sketch.addGeometry(line)

# 隐藏草图以便于观察后续步骤（可选）
# Gui.ActiveDocument.getObject('Profile').Visibility = False

# --- 4. 旋转成型 (Revolution) ---
# 使用 PartDesign 的旋转功能生成实体
revolution = App.ActiveDocument.addObject('PartDesign::Revolution', 'VaseRevolution')
revolution.Profile = sketch
revolution.Angle = 360.0  # 旋转 360 度
revolution.Axis = (0, 1, 0) # 绕草图的 Y 轴 (垂直轴) 旋转
revolution.Refine = True # 自动优化边缘

# --- 5. 设置颜色 (赤陶土色) ---
revolution.ViewObject.ShapeColor = (0.85, 0.4, 0.2) 

# --- 6. 更新文档并调整视图 ---
doc.recompute()

# 适应视图
Gui.SendMsgToActiveView("ViewFit")

# 设置为等轴测视图 (Isometric View)
Gui.activeDocument().activeView().viewIsometric()

print("花瓶已成功创建！")
```
