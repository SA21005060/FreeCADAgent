这是一个完整的 FreeCAD Python 宏代码，它将创建一个球体，调整视图以适应球体，并将其设置为等轴测（Isometric）视图。

### FreeCAD 宏代码

您可以直接复制以下代码并在 FreeCAD 中运行：

```python
import FreeCAD
import FreeCADGui

# 1. 创建球体 (Part::Sphere)
# 参数 "MySphere" 是您在对象树中看到的名称
sphere = FreeCAD.ActiveDocument.addObject("Part::Sphere", "MySphere")

# 2. 设置球体半径 (单位通常为 mm)
sphere.Radius = 10

# 3. 重新计算文档，确保几何体生成
FreeCAD.ActiveDocument.recompute()

# 4. 缩放视图以适应创建的球体
FreeCADGui.SendMsgToActiveView("ViewFit")

# 5. 设置为等轴测视图 (Isometric View)
FreeCADGui.ActiveDocument.activeView().viewIsometric()
```

### 如何运行此代码：

1.  **方法一：宏菜单**
    *   打开 FreeCAD。
    *   点击顶部菜单 **宏 (Macro)** -> **宏... (Macros...)**。
    *   点击 **创建 (Create)** (或直接点击 **执行 (Execute)** 如果您已经保存了文件)。
    *   将代码粘贴到弹出的编辑器中，按 **F5** 执行。

2.  **方法二：Python 控制台**
    *   在 FreeCAD 界面中，按 `Ctrl + Shift + P` 打开 **Python 控制台**。
    *   将代码粘贴到控制台中并按 **Enter** 键。

### 代码说明：
*   `addObject("Part::Sphere", "MySphere")`: 创建一个球体几何特征。
*   `sphere.Radius = 10`: 将球体的半径设置为 10 毫米（您可以修改这个值）。
*   `viewIsometric()`: 这是一个内置函数，可以立即将相机角度设置为标准的等轴测视图。
