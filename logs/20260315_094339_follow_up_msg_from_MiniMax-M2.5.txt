这是一个使用 FreeCAD Python 创建“花瓶”的完整宏。

这个脚本使用了**旋转 (Revolution)** 和 **B样条曲线 (BSpline)** 技术。它通过定义一系列的半径和高度点来绘制花瓶的轮廓，然后通过旋转这些点来创建一个平滑的、带有中空内壁的花瓶。

### 使用方法：
1. 打开 FreeCAD。
2. 转到菜单 **宏** -> **宏...**。
3. 点击 **创建**。
4. 输入名称（例如 `Vase`），然后将下面的代码粘贴进去。
5. 点击绿色的 “Play” 按钮（执行宏）。

### Python 代码：

```python
import FreeCAD
import Part
import math

# ==========================================
# 1. 设置文档
# ==========================================
# 如果没有打开文档，则创建一个新的
if not FreeCAD.ActiveDocument:
    FreeCAD.newDocument()

doc = FreeCAD.ActiveDocument

# ==========================================
# 2. 定义花瓶的轮廓点 (高度, 半径)
# ==========================================
# 外轮廓点 (构成花瓶的外部形状)
# 格式: FreeCAD.Vector(半径, 高度, 0)
points_outer = [
    FreeCAD.Vector(20, 0, 0),    # 底部中心
    FreeCAD.Vector(30, 5, 0),    # 底部边缘
    FreeCAD.Vector(60, 40, 0),   # 瓶身最宽处
    FreeCAD.Vector(40, 80, 0),   # 瓶颈弯曲处
    FreeCAD.Vector(25, 110, 0),  # 瓶颈
    FreeCAD.Vector(30, 120, 0)   # 瓶口边缘
]

# 内轮廓点 (构成花瓶的内壁，用于挖空)
# 这些点比外轮廓稍微小一点，形成厚度
points_inner = [
    FreeCAD.Vector(15, 5, 0),    # 底部内凹
    FreeCAD.Vector(25, 10, 0),   # 内壁底部
    FreeCAD.Vector(55, 40, 0),   # 瓶身内部
    FreeCAD.Vector(35, 80, 0),   # 瓶颈内部
    FreeCAD.Vector(20, 110, 0),  # 颈部内部
    FreeCAD.Vector(25, 120, 0)   # 瓶口内部
]

# ==========================================
# 3. 创建 B样条曲线 (平滑轮廓)
# ==========================================
# 创建外轮廓样条曲线
spline_outer = Part.BsplineCurve()
spline_outer.interpolate(points_outer)
shape_outer = spline_outer.toShape()
wire_outer = Part.Wire(shape_outer.Edges)

# 创建内轮廓样条曲线
spline_inner = Part.BsplineCurve()
spline_inner.interpolate(points_inner)
shape_inner = spline_inner.toShape()
wire_inner = Part.Wire(shape_inner.Edges)

# ==========================================
# 4. 创建面并旋转
# ==========================================
# 创建一个带有“孔洞”的面 (第一个是外轮廓，第二个是内轮廓/孔)
# 注意：这里不需要手动处理线的方向，FreeCAD会自动识别
face = Part.Face([wire_outer, wire_inner])

# 定义旋转轴 (Z轴)
axis = FreeCAD.Vector(0, 0, 1)

# 旋转 360 度生成实体
vase = face.revolve(axis, 360)

# ==========================================
# 5. 显示结果
# ==========================================
Part.show(vase)
doc.recompute()

# ==========================================
# 6. 视图设置 (等轴测视图并适应模型)
# ==========================================
Gui = FreeCADGui
Gui.activeDocument().activeView().viewIsometric()
Gui.activeDocument().activeView().fitAll()
```

### 代码解析：
1.  **定义轮廓 (`points_outer` & `points_inner`)**:
    *   我们定义了花瓶的“半边”截面。
    *   外层点定义了花瓶的外部形状（底部宽 -> 瓶身宽 -> 颈部窄 -> 瓶口宽）。
    *   内层点定义了内部空腔，点的数值比外层稍小，从而决定了花瓶的壁厚。
2.  **B样条曲线 (BsplineCurve)**:
    *   使用普通的直线连接这些点会显得很僵硬。`Part.BsplineCurve` 会在这些点之间生成平滑的曲线，让花瓶看起来像真正的车削陶瓷或车塑作品。
3.  **旋转 (Revolution)**:
    *   `face.revolve(axis, 360)` 将2D的平面截面沿着Z轴旋转一周，瞬间生成3D实体。
4.  **视图设置**:
    *   代码最后自动将视角切换到等轴测（Isometric）并自动缩放视图，方便你直接查看结果。

你可以通过修改 `points_outer` 和 `points_inner` 列表中的数字来改变花瓶的形状（例如更细、更胖、更高）。
