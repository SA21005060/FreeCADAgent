这是一个为 FreeCAD 编写的 Python 宏脚本。它将创建一个包含瓶身、瓶颈和瓶盖的基本水瓶模型，并自动调整视角为等轴测（Isometric）视图。

### 使用方法
1. 打开 FreeCAD 软件。
2. 在菜单栏中找到 **“宏” (Macro)** -> **“宏...” (Macros...)**。
3. 在弹出的对话框中，点击 **“创建” (Create)** 按钮。
4. 将下面的代码完整复制并粘贴到编辑器中。
5. 点击工具栏上的 **“保存” (Save)** 按钮保存（可选）。
6. 点击 **“执行” (Execute)** 按钮运行脚本。

### Python 宏代码

```python
import FreeCAD
import Part
import FreeCADGui
import math

# --- 1. 准备文档 ---
# 确保当前有活动文档，如果没有则新建一个
if not FreeCAD.ActiveDocument:
    FreeCAD.newDocument()

# 获取活动文档
doc = FreeCAD.ActiveDocument

# 清理当前视图中的旧对象（方便重复运行脚本）
for obj in doc.Objects:
    doc.removeObject(obj.Name)

# --- 2. 定义参数 (单位: mm) ---
body_radius = 30
body_height = 100

neck_radius = 15
neck_height = 30

cap_radius = 16
cap_height = 15

# --- 3. 创建几何体 ---

# A. 创建瓶身 (圆柱体)
# Part.makeCylinder(半径, 高度)
body = Part.makeCylinder(body_radius, body_height)

# B. 创建瓶颈 (圆柱体)
neck = Part.makeCylinder(neck_radius, neck_height)
# 将瓶颈移动到瓶身顶部
# translate 需要一个 FreeCAD.Vector(x, y, z)
neck.translate(FreeCAD.Vector(0, 0, body_height))

# C. 创建瓶盖 (圆柱体)
cap = Part.makeCylinder(cap_radius, cap_height)
# 将瓶盖移动到瓶颈顶部
cap.translate(FreeCAD.Vector(0, 0, body_height + neck_height))

# --- 4. 合并形状 ---
# 使用 fuse (融合) 将三个形状合并为一个实体
bottle_shape = body.fuse(neck).fuse(cap)

# --- 5. 显示并更新 ---
# 在模型树中创建一个对象并显示该形状
Part.show(bottle_shape)
doc.recompute()

# --- 6. 视角设置 ---
# 调整视图以适应模型 (Zoom to Fit)
FreeCADGui.SendMsgToActiveView("ViewFit")

# 设置为等轴测视图 (Isometric View)
# 注意：需要确保 FreeCAD 处于 GUI 模式
if FreeCADGui.ActiveDocument:
    FreeCADGui.ActiveDocument.ActiveView.viewIsometric()
```

### 代码说明：
1.  **几何构建**：脚本使用 `Part.makeCylinder` 创建了三个圆柱体分别代表瓶身、瓶颈和瓶盖。
2.  **定位 (Translation)**：由于 FreeCAD 的原点默认在 (0,0,0)，我们使用 `translate` 函数将瓶颈和瓶盖沿着 Z 轴向上移动，使其叠加在瓶身之上。
3.  **布尔运算 (Boolean)**：使用 `.fuse()` 方法将三个独立的圆柱体融合在一起，形成一个连续的水瓶实体。
4.  **视角控制**：
    *   `ViewFit`：自动计算最佳缩放比例，让整个模型充满视图。
    *   `viewIsometric()`：将相机旋转到标准的等轴测角度（立体图视角），方便查看 3D 效果。
