Here is the corrected macro.

The main issue in the previous code was that `App` and `Gui` were used without defining them. In FreeCAD macros, you must explicitly assign `App = FreeCAD` and `Gui = FreeCAD.Gui` (or use the full names).

I have also slightly adjusted the positions of the spout and handle to ensure they look connected to the body, and added a simple lid for better aesthetics.

```python
import FreeCAD
import Part
import math

# Define aliases to ensure 'App' and 'Gui' are available in the global scope
App = FreeCAD
Gui = FreeCAD.Gui

# 1. Create new document
App.newDocument("Kettle")
App.setActiveDocument("Kettle")
App.ActiveDocument = App.getDocument("Kettle")
Gui.ActiveDocument = App.getDocument("Kettle")

# 2. Body: Main cylinder
# Center at (0,0,0). Height 120, Radius 40.
body = App.ActiveDocument.addObject("Part::Cylinder", "Body")
body.Radius = 40
body.Height = 120
body.Placement = App.Placement(App.Vector(0, 0, 0), App.Rotation(0, 0, 0, 1))

# 3. Lid: Cylinder on top
# Radius slightly smaller (38), Height 10.
# Top of body is at z=60. Lid center at z=55.
lid = App.ActiveDocument.addObject("Part::Cylinder", "Lid")
lid.Radius = 38
lid.Height = 10
lid.Placement = App.Placement(App.Vector(0, 0, 55), App.Rotation(0, 0, 0, 1))

# 4. Spout: Cylinder on the side
# Radius 10, Height 40.
# Positioned at x=40 (surface of body), z=20.
# Rotated -45 degrees around X axis to point up and right.
spout = App.ActiveDocument.addObject("Part::Cylinder", "Spout")
spout.Radius = 10
spout.Height = 40
spout.Placement = App.Placement(App.Vector(40, 0, 20), App.Rotation(App.Vector(1, 0, 0), -45))

# 5. Handle: Torus
# Radius1 (Major) = 30, Radius2 (Minor) = 6.
# Rotated 90 degrees around Y to make it a vertical loop.
# Positioned at x=-40.
handle = App.ActiveDocument.addObject("Part::Torus", "Handle")
handle.Radius1 = 30
handle.Radius2 = 6
handle.Placement = App.Placement(App.Vector(-40, 0, 30), App.Rotation(App.Vector(0, 1, 0), 90))

# 6. Recompute
App.ActiveDocument.recompute()

# 7. Set View
Gui.ActiveDocument.activeView().viewIsometric()
Gui.ActiveDocument.activeView().fitAll()
```
