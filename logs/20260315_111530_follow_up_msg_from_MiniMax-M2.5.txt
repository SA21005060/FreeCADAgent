这是一个在 FreeCAD 中创建球体的 Python 宏。

这个脚本会：
1. 创建一个球体（默认为半径 5）。
2. 将其添加到活动文档中。
3. 调整视图以适应球体并设置为等轴测视图（Isometric View）。

```python
import FreeCAD
import Part
import FreeCADGui

# 定义球体的半径
radius = 5.0

# 创建球体形状
# 语法: Part.makeSphere(radius, [pnt, dir, angle1, angle2, angle3])
sphere = Part.makeSphere(radius)

# 获取或创建活动文档
doc = FreeCAD.ActiveDocument
if doc is None:
    doc = FreeCAD.newDocument("Sphere_Doc")

# 将球体形状添加为 Part::Feature 对象
sphere_obj = doc.addObject("Part::Feature", "MySphere")
sphere_obj.Shape = sphere

# 重新计算文档以更新显示
doc.recompute()

# 视图设置
if FreeCAD.GuiUp:
    # 获取当前视图
    view = FreeCADGui.ActiveDocument.ActiveView
    
    # 缩放视图以适应模型
    view.fitAll()
    
    # 设置为等轴测视图 (Isometric View)
    view.viewIsometric()
```

### 如何使用：
1. 打开 FreeCAD。
2. 转到菜单 **宏** -> **宏...** (Macro -> Macros)。
3. 点击 **创建** (Create) 按钮。
4. 将上述代码粘贴进去并保存。
5. 点击 **执行** (Execute) 按钮运行宏。
