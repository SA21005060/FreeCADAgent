**错误分析：**
之前的代码报错 `'NoneType' object has no attribute 'addObject'`，这是因为在运行代码时 FreeCAD 中没有打开任何文档（`FreeCAD.ActiveDocument` 为 `None`），因此无法向其中添加对象。

**解决方案：**
在创建长方体之前，添加一段检查代码。如果当前没有活动文档，则自动创建一个新文档。

**修复后的代码如下：**

```python
import FreeCAD
import FreeCADGui

# ==========================================
# 1. 检查并创建活动文档 (修复 NoneType 错误)
# ==========================================
if FreeCAD.ActiveDocument is None:
    FreeCAD.newDocument()

# ==========================================
# 2. 定义长方体的尺寸 (单位：mm)
# ==========================================
length = 50  # 长度 (X轴方向)
width = 30   # 宽度 (Y轴方向)
height = 20  # 高度 (Z轴方向)

# ==========================================
# 3. 创建长方体对象
# ==========================================
# 使用 Part 工作台的几何体功能创建 Box
box = FreeCAD.ActiveDocument.addObject("Part::Box", "MyBox")
box.Length = length
box.Width = width
box.Height = height

# 确保属性设置后几何体正确生成
FreeCAD.ActiveDocument.recompute()

# ==========================================
# 4. 设置视图并适应屏幕
# ==========================================
# 获取当前视图对象
view = FreeCADGui.ActiveDocument.ActiveView

# 设置为等轴测视图 (Isometric View)
view.viewIsometric()

# 自动缩放视图以完全显示创建的物体
view.fitAll()

print("长方体已创建，尺寸: {} x {} x {}".format(length, width, height))
```
