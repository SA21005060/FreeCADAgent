这是一个在 FreeCAD 中创建长方体（立方体/盒子）的 Python 宏脚本。

这段代码会创建一个指定尺寸的长方体，并将视图自动调整为等轴测（Isometric）视角且自动缩放以适应屏幕。

```python
import FreeCAD
import FreeCADGui

# ==========================================
# 1. 定义长方体的尺寸 (单位：mm)
# ==========================================
length = 50  # 长度 (X轴方向)
width = 30   # 宽度 (Y轴方向)
height = 20  # 高度 (Z轴方向)

# ==========================================
# 2. 创建长方体对象
# ==========================================
# 使用 Part 工作台的几何体功能创建 Box
box = FreeCAD.ActiveDocument.addObject("Part::Box", "MyBox")
box.Length = length
box.Width = width
box.Height = height

# 确保属性设置后几何体正确生成
FreeCAD.ActiveDocument.recompute()

# ==========================================
# 3. 设置视图并适应屏幕
# ==========================================
# 获取当前视图对象
view = FreeCADGui.ActiveDocument.ActiveView

# 设置为等轴测视图 (Isometric View)
view.viewIsometric()

# 自动缩放视图以完全显示创建的物体
view.fitAll()

print("长方体已创建，尺寸: {} x {} x {}".format(length, width, height))
```

### 如何使用：
1. 打开 FreeCAD。
2. 点击菜单栏的 **宏** (Macros) -> **执行宏** (Execute Macro)。
3. 在弹出的对话框中，粘贴上面的代码并点击“执行”。
4. 或者，您可以将代码保存为 `.py` 文件（例如 `make_box.py`），然后从宏管理器中运行它。

### 代码说明：
*   **`Part::Box`**：这是 FreeCAD 中用于创建长方体的核心对象类型。
*   **`Length`, `Width`, `Height`**：定义长方体的三个维度。
*   **`viewIsometric()`**：将相机视角切换为工程常用的等轴测视图。
*   **`fitAll()`**：计算并调整缩放比例，确保创建的长方体完整地显示在当前视图窗口中。
